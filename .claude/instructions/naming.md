# Naming Conventions

## Blog Posts & URLs

- **Filename:** `kebab-case.mdx` → `/blog/kebab-case/`
- **Title:** Spanish, clear (used in nav + metadata)
- **Category:** Exact enum match (see below)
- **Tags:** lowercase, kebab-case, Spanish-friendly

**Valid categories (enforce enum in schema):**
- `Modelos`
- `Inteligencia Artificial`
- `Conceptos`
- `Arquitectura`
- `Herramientas`
- `Ética`
- `Tendencias`

## Assets

- **Hero images:** `source/assets/post/{filename}.png`
- **OG images:** Auto-generated (don't commit)
- **SVGs:** `source/assets/icons/` if reusable

Name by purpose: `og-{post-slug}.png`, `hero-{post-slug}.png`

## Data Structures

- **Knowledge map:** `source/data/generated/knowledge-map.ts`
- **Post insights:** `source/data/generated/post-insights.ts`
- **Editorial radar:** `source/data/generated/editorial-radar.ts`

Regenerated by `npm run build:data`. Never edit.

## Routes

- **Blog:** `/blog/{slug}/`
- **Tags:** `/tag/{slug}/` (dead tags = 410 Gone)
- **Category:** `/category/{slug}/`
- **Pages:** `/mapa-ia/`, `/radar/`, `/metricas/`
- **API:** `/api/search-console/report.json`

Trailing slashes required. Redirects for old routes.

## Component Props

```typescript
interface PostCardProps {
  title: string;
  slug: string;
  generatedBy: string;  // LLM that created it
  tags: string[];
  category: string;
}
```

No abbreviations. `generatedBy`, not `genBy`.

## Environment Variables

- **Local:** `.dev.vars` (gitignored)
- **Secrets:** Cloudflare `wrangler secret put`
- **Naming:** `GOOGLE_SEARCH_CONSOLE_TOKEN`, `SATORI_API_KEY`

UPPER_SNAKE_CASE, descriptive.
